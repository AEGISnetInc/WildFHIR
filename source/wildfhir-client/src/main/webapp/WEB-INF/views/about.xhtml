This is the AEGIS WildFHIR Community Edition Reference Implementation Test Client web application. The interfaces provide the following functionality:

  • FHIR Interactions - Send valid FHIR interactions to a selected FHIR server
    • read - return a FHIR resource instance by its resource id
    • vread - return a FHIR resource instance by its resource id and version id
    • history - return the FHIR history by global, resource type or resource instance
    • search - return FHIR search results for a resource type
    • create - send a FHIR resource to be created
    • update - send a FHIR resource to be updated
    • patch - send a JSON or XML patch and FHIR resource to be updated
    • metadata - return a selected server's CapabilityStatement
  • FHIR Operations - Send FHIR extended operations to a selected FHIR server
    • $convert - FHIR extended operation to convert JSON to/from XML
    • $everything - FHIR everything operation
    • $validate - validate FHIR resources (validation is supported by the underlying HL7 FHIR Core Library and pre-loaded FHIR R4 v4.0.1 validation packages)
  • Tools - WildFHIR Tools
    • FHIR Path - execute FHIR Path evaluations using the AEGIS WildFHIR extended operation '$fhirpath-evaluate
    • Subscriptions - manually execute the Subscription Service for the AEGIS WildFHIR Community Edition server
  • Admin - Administration functionality
    • Servers - manage the list of known FHIR Servers and their base url value
